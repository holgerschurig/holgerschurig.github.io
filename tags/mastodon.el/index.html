mastodon.el: a Hydra to memorize/access it's many commands

0001-01-01
Emacs
Emacs / mastodon.el

Mastodon has a lot of commands. You can access most of them directly via keybindings, just use M-x describe-mode or the h or ? keys. Some more are described on the project website, https://codeberg.org/martianh/mastodon.el in the README.org But, if you don’t use Mastodon that regularly, maybe a Hydra can help you?
